# ESP32-S3 433MHz 智能遥控器 - 固件烧录说明

## 📦 固件文件

- **固件版本**: v2.0.0
- **固件文件**: `ESP32-S3-433MHz-v2.0.0.bin`
- **适用开发板**: ESP32-S3系列（推荐立创ESP32-S3-R8N8）
- **固件大小**: 约752KB

## 🔧 烧录方法

### 方法一：使用 esptool.py（推荐）

#### 1. 安装 esptool

```bash
pip install esptool
```

#### 2. 烧录固件

**Windows:**
```bash
esptool.py --chip esp32s3 --port COM3 --baud 921600 write_flash 0x10000 ESP32-S3-433MHz-v2.0.0.bin
```

**macOS/Linux:**
```bash
esptool.py --chip esp32s3 --port /dev/ttyUSB0 --baud 921600 write_flash 0x10000 ESP32-S3-433MHz-v2.0.0.bin
```

> 💡 **注意**: 请将 `COM3` 或 `/dev/ttyUSB0` 替换为您的实际端口号

#### 3. 查看端口号

**Windows**: 设备管理器 → 端口(COM和LPT)

**macOS**: 
```bash
ls /dev/cu.usbserial-* 或 ls /dev/cu.usbmodem*
```

**Linux**: 
```bash
ls /dev/ttyUSB* 或 ls /dev/ttyACM*
```

### 方法二：使用 Flash Download Tools（Windows图形化工具）

#### 1. 下载工具
- 官方下载: https://www.espressif.com/zh-hans/support/download/other-tools
- 选择 "Flash Download Tools"

#### 2. 配置参数
1. 选择芯片类型: **ESP32-S3**
2. 选择固件文件: `ESP32-S3-433MHz-v2.0.0.bin`
3. 烧录地址: `0x10000`
4. 选择COM口
5. 波特率: `921600`
6. 点击 **START** 开始烧录

### 方法三：使用 PlatformIO（开发者）

如果您已经安装了PlatformIO：

```bash
# 克隆项目
git clone https://github.com/zhoushoujianwork/433_test_arduino.git
cd 433_test_arduino

# 直接烧录
pio run -t upload
```

## ⚡ 烧录步骤详解

### 1. 连接开发板
- 使用USB数据线连接ESP32-S3开发板到电脑
- 确保驱动已正确安装

### 2. 进入下载模式（如果烧录失败）
1. 按住 **BOOT** 按钮不放
2. 按一下 **RESET** 按钮
3. 松开 **BOOT** 按钮
4. 开始烧录

### 3. 烧录完成
- 看到 "Hash of data verified" 表示烧录成功
- 按一下 **RESET** 按钮重启设备
- 或断电重新上电

## 🔍 常见问题

### Q: 烧录失败，提示连接错误？
**A**: 
1. 检查USB线是否正常（建议使用数据线，不要用充电线）
2. 检查驱动是否安装（CH340或CP2102驱动）
3. 尝试进入下载模式后再烧录
4. 降低波特率到 `115200` 重试

### Q: 烧录成功但设备无反应？
**A**: 
1. 按一下RESET按钮重启
2. 检查串口监视器是否有输出
3. 确认固件地址是 `0x10000`

### Q: 如何查看串口输出？
**A**: 
使用串口工具（波特率115200）：
- Arduino IDE: 工具 → 串口监视器
- PlatformIO: `pio device monitor`
- 其他工具: PuTTY, minicom等

### Q: 如何恢复出厂设置？
**A**: 
重新烧录固件即可，或长按Boot按钮2秒清空信号。
