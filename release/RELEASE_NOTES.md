# Release Notes - v2.0.0

## 🎉 ESP32-S3 433MHz 智能遥控器 v2.0.0

**发布日期**: 2025-11-08

这是一个重大版本更新，带来了全新的WiFi Web管理界面和多信号管理功能！

---

## ✨ 新增功能

### 📱 WiFi Web管理界面
- **WiFi AP模式**: 开发板自动创建WiFi热点（SSID: ESP433RF）
- **手机端控制**: 无需电脑，手机浏览器即可管理
- **iPhone风格UI**: 圆角设计，渐变色按钮，操作流畅
- **响应式布局**: 完美适配各种屏幕尺寸

### 🔄 多信号管理系统
- **存储50个信号**: 支持存储最多50个不同的遥控信号
- **智能去重**: 自动识别重复信号，避免重复学习
- **Flash持久化**: 所有信号保存在Flash中，断电不丢失
- **信号列表**: 最新学习的信号自动显示在最上面

### 🎯 Boot按钮绑定
- **一键绑定**: 将常用信号绑定到物理Boot按钮
- **快速发送**: 短按Boot按钮即可发送绑定的信号
- **Web管理**: 在网页上轻松绑定/解绑信号

### 🛡️ 安全与规范
- **使用声明**: 多处显著提示，明确使用规范
- **法律声明**: 完善的免责条款和法律责任说明
- **技术规范**: 符合国家无线电管理规定（≤10mW）

## 🔧 功能优化

### 学习模式改进
- ✅ 一次只学习一个信号，学习后自动退出
- ✅ 自动去重，相同信号不会重复添加
- ✅ 学习完成后立即显示在列表最上方

### 发送优化
- ✅ 发送时自动禁用接收，避免自干扰
- ✅ 支持Web发送和物理按键发送两种方式
- ✅ 发送状态实时反馈

### 用户体验
- ✅ Toast提示替代弹窗，2秒自动消失
- ✅ 清晰的状态指示（LED + 串口输出）
- ✅ 简化操作流程，新手友好

---

## 🗑️ 移除功能

- ❌ **串口命令交互**: 移除所有串口命令（test, capture, send等）
- ✅ **保留串口输出**: 调试信息仍通过串口输出

> 💡 **原因**: 通过Web界面控制更直观方便，简化使用流程

## 📚 技术架构

### 新增库
- **SignalManager**: 信号管理库，负责存储和管理多个信号
- **ESP433RFWeb**: Web服务器库，提供WiFi和HTTP服务

### ESP433RF库增强
- 新增接收控制API: `enableReceive()` / `disableReceive()` / `isReceiving()`
- 优化信号接收逻辑

### 系统架构
- **FreeRTOS多任务**: 接收、状态、按钮、LED独立任务
- **WiFi AP模式**: 192.168.4.1 固定IP
- **RESTful API**: JSON格式数据交互

## 🛠️ 硬件要求

### 推荐配置
- **开发板**: 立创ESP32-S3-R8N8（8MB Flash + 8MB PSRAM）
- **接收模块**: **灵-R1A-M5_433** (串口版本) ⚠️ 必须选择M5串口版本
- **发射模块**: MX-FS-03V 或 Ling-T3Max
- **LED指示灯**: 可选

> ⚠️ **重要提示**：接收模块必须选择 **灵-R1A-M5_433/315 串口版本**，不要选择普通版本！  
> 串口版本的接收模块可以输出数字信号，普通版本无法正常工作。

### 引脚连接
```
ESP32-S3-R8N8
├── GPIO14 → 发射模块DATA (MX-FS-03V 或 Ling-T3Max)
├── GPIO18 → 接收模块DATA (灵-R1A-M5_433 串口版本)
├── GPIO21 → LED（可选）
└── GPIO0  → Boot按钮（板载）
```

## 🚀 快速开始

### 1. 烧录固件
```bash
esptool.py --chip esp32s3 --port COM3 --baud 921600 \
  write_flash 0x10000 ESP32-S3-433MHz-v2.0.0.bin
```

### 2. 连接WiFi
- WiFi名称: `ESP433RF`
- WiFi密码: `12345678`

### 3. 打开管理界面
- 访问: `http://192.168.4.1`

### 4. 学习信号
1. 点击"捕获信号"按钮
2. 按下遥控器按键
3. 信号自动保存

### 5. 使用信号
- 网页点击"发送"按钮
- 或绑定到Boot按钮快速发送
